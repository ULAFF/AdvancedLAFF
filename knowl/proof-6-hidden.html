If \(x = 0 \text{,}\) the result obviously holds since then \(\| A x \| = 0 \) and \(\| x \| = 0 \text{.}\) Let \(x \neq 0 \text{.}\) Then

\begin{equation*} \| A \| = \max_{x \neq 0} \frac{\| A x \|} {\| x \|} \geq \frac{\| A x \|} {\| x \|}. \end{equation*}

Rearranging this yields \(\| A x \| \leq \| A \| \| x \| \text{.}\)
