Let \(x \in \C^n \) and \(\vec{0} \) the zero vector of size \(n \) and \(0 \) the scalar zero. Then

\begin{equation*} \begin{array}{l} \nu( \vec 0 ) \\ ~~~= ~~~~( 0 \cdot x = \vec 0 ) \\ \nu( 0 \cdot x ) \\ ~~~=~~~~{\rm is~homogeneous} \\ 0 \nu( x ) \\ ~~~=~~~( 0 \cdot x = \vec 0 ) \\ ~~~ = ({\rm algebra}) 0 \end{array} \end{equation*}
